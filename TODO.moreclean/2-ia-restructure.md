# 2 — IA restructure: nav, pages, actions link, harmonization rename

## Problem
Nav doesn't map to TC1's mental model. TC1's major issues are:
1. **ID conflicts** — same G18 number → different concepts (numbering error)
2. **Definition conflicts** — same term, divergent definitions across pubs

These are the two PROBLEMS TC1 must solve. Everything else is lookup/reference.

## Proposed IA
```
Actions          → overview of what TC1 needs to do (priority worklist + tools)
ID Conflicts     → numbering errors (same ID, different concepts)
Defn Conflicts   → divergent definitions across pubs (was "Harmonise")
Terms            → browse/search all terms (lookup)
Publications     → browse all publications (lookup)
TC / SC          → browse by committee (lookup)
Editions         → 2010 vs 202X comparison
```

## Changes
1. Rename "Harmonise" → "Definition Conflicts" in nav + page title.
2. Add clearer explanatory copy on the Definition Conflicts page.
3. Remove redundant "Open →" column from Actions page (term name is
   already a link).
4. Regroup nav visually: Problems (ID + Defn Conflicts) | Lookup (Terms,
   Pubs, TC/SC) | Reference (Editions) — Actions stays first.

## Nav structure
```
Actions | ID Conflicts | Defn Conflicts  ·  Terms | Publications | TC/SC  ·  Editions
```
